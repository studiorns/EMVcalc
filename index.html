What is EMV?

Earned Media Value (EMV) is a metric that quantifies the value of exposure gained through promotional efforts. This calculator uses a formula based on impressions and engagements, adjusted for market and tier factors.

Base vs. Adjusted EMV

Base EMV is calculated using only impressions and engagements (0.014 × Impressions + 0.14 × Engagements + 7000), providing a standard value regardless of market or influencer tier.

Adjusted EMV is the final value after applying market and tier multipliers to the base EMV (Base EMV × Market Multiplier × Tier Multiplier), giving a more tailored estimate for specific markets and influencer tiers.

EMV Calculation Methodology

EMV Formula

The base EMV formula is:

EMV = 0.014 × Impressions + 0.14 × Engagements + 7000

This base formula is then adjusted using multipliers specific to each market and tier:

Adjusted EMV = Base EMV × Market Multiplier × Tier Multiplier

Additional Metrics

  • EMV per 1,000 Impressions = (Adjusted EMV / Impressions) × 1,000
  • EMV per 1,000 Engagements = (Adjusted EMV / Engagements) × 1,000
  • EMV Efficiency Ratio = Adjusted EMV / (Impressions + Engagements)

Example Calculation

Let's walk through an example calculation for a Tier 2 influencer in the United Kingdom:

Step 1: Input Values

  • Impressions: 5,000,000
  • Engagements: 100,000
  • Market: United Kingdom (Multiplier: 1.30)
  • Tier: Tier 2 (Multiplier: 1.10)

Step 2: Calculate Base EMV

Base EMV = 0.014 × Impressions + 0.14 × Engagements + 7000
Base EMV = 0.014 × 5,000,000 + 0.14 × 100,000 + 7000
Base EMV = 70,000 + 14,000 + 7000
Base EMV = $91,000

Step 3: Apply Market Multiplier

Market Adjusted EMV = Base EMV × Market Multiplier
Market Adjusted EMV = $91,000 × 1.30
Market Adjusted EMV = $118,300

Step 4: Apply Tier Multiplier

Final Adjusted EMV = Market Adjusted EMV × Tier Multiplier
Final Adjusted EMV = $118,300 × 1.10
Final Adjusted EMV = $130,130

Step 5: Calculate Additional Metrics

EMV per 1,000 Impressions = (Adjusted EMV / Impressions) × 1,000
EMV per 1,000 Impressions = ($130,130 / 5,000,000) × 1,000
EMV per 1,000 Impressions = $26.03

EMV per 1,000 Engagements = (Adjusted EMV / Engagements) × 1,000
EMV per 1,000 Engagements = ($130,130 / 100,000) × 1,000
EMV per 1,000 Engagements = $1,301.30

EMV Efficiency Ratio = Adjusted EMV / (Impressions + Engagements)
EMV Efficiency Ratio = $130,130 / (5,000,000 + 100,000)
EMV Efficiency Ratio = $130,130 / 5,100,000
EMV Efficiency Ratio = 0.0255
